Lumi

Your Friendly Discord Companion

Lumi is a versatile Discord bot designed to brighten up your server with fun commands, moderation tools, and helpful utilities. With a cute interface and powerful features, Lumi is the perfect addition to any Discord community.

Fun Commands

Enjoy a variety of fun commands to keep your server entertained! From memes and games to cute animal pictures, Lumi has it all.

Moderation Tools

Keep your server safe with powerful moderation tools. Manage users, delete messages, and set up auto-moderation with ease.

Custom Responses

Create custom commands and responses tailored to your server's unique needs and culture.

Popular Commands

!help

Displays a list of all available commands and how to use them.

!emote (emote) [@user-mention]

Use funny emote on someone such and kick and more

/petpet

Pet anyone you want with their user id!

!weather [location]

Gets the current weather forecast for any location. (coming soon!)
